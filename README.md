Setup
-----

[Node.js](http://nodejs.org/) and [Redis](http://redis.io) are required to use this app.

Install dependencies via `package.json`.

```bash
$ npm install
```

Shoot the Node server.

```bash
$ node server.js
```

Play in Browser.

```text
http://localhost:8080
```
